## Instructions for Developer Agents

When modifying this repository, future developer agents should follow these rules:
1. **Maintain Theme Purity**: Keep the matte black background (`#050505`) and glowing matrix green theme (`#00FF00`).
2. **Keep the Monospaced Flow**: Use monospaced fonts (`JetBrains Mono`, `Fira Code`) for headers, tags, and data logs.
3. **Local Dev Server**: Always ensure the dev server script (`npm run dev`) runs successfully when validating code changes.
4. **Verification**: Always run verification before concluding a task to confirm changes display properly in the browser environment.
